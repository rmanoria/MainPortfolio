# Akinwale Emmanuel — Developer Portfolio

A dark, high-performance personal portfolio built with HTML, CSS, and vanilla JavaScript.

## Features
- ✦ Animated particle network hero canvas
- ✦ Typewriter role switcher
- ✦ Custom cursor with magnetic lag effect
- ✦ Scroll-triggered reveal animations
- ✦ SVG skill rings with animated fill
- ✦ 3D tilt effect on project cards
- ✦ Responsive down to mobile
- ✦ JetBrains Mono + Space Grotesk typography

## Deployment

Deployed on [Vercel](https://vercel.com). Every push to `main` triggers a new deployment.

## Local Preview

Just open `index.html` in a browser — no build step required.
